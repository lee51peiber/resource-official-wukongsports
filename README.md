# resource-official-wukongsports

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自构成完整的内容单元，不依赖于特定域名或外部网站结构。

## 项目简介

这是一个以资源归档为目的的仓库，主要存放静态 HTML 文件。每个页面均为独立完整的实体，可直接在浏览器中打开查看。仓库不绑定任何具体网站或线上服务，仅作为页面资料存档与分发用途。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库入口页面（可选）
├── pages/             # 存放独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 公共资源文件（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：核心目录，所有独立 HTML 页面存放于此。每个文件可独立运行。
- `assets/`：页面间共享的静态资源，按类别组织。

## 页面归档说明

归档的 HTML 页面具有以下特点：

- **自包含**：每个页面包含完整的内容与样式，不依赖外部服务。
- **离线可用**：页面所需资源尽量本地化，确保在无网络环境下也可正常浏览。
- **版本化**：更新时保留历史版本（通过 Git 提交记录），便于回溯。

### 如何添加新页面

1. 将 HTML 文件放入 `pages/` 目录。
2. 如有共享资源，请放入 `assets/` 对应子目录。
3. 在 `index.html`（如有）中添加链接。
4. 提交 Pull Request 或直接推送。
